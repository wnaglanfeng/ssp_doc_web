# 创建媒体获取appKey

## 1. 登录管理系统

按照我们的商务同学提供的登录凭证，首先【[登录亘元有量管理系统](https://admin.ssp.igenyuan.com/login)】。

## 2. 创建媒体账号

- 登录后，点击【媒体管理】。
- 点击【创建媒体】。
![创建媒体](/images/sdk/to_create_media.png)
- 填写媒体信息，包括【媒体名称】、【媒体类型】、【媒体描述】等。
![填写媒体信息](/images/sdk/create_media_dialog.png)
- 点击【确定】。

## 3. 获取`appKey`
创建成功后，在媒体列表中会显示`appKey`，请保存好，后续接入时需要使用。
![appKey](/images/sdk/app_key.png)


***注意：***
- `appKey`是您的媒体账号唯一标识符，后续接入时需要使用。
- 请妥善保管，避免泄露给他人。

## 4. 媒体状态
- 媒体状态分为`待上线`和`已上线`两种。
- `待上线`状态的媒体账号，如有需要请联系商务同学进行审核，审核通过后才可以进行对接。
- `已上线`状态的媒体账号，会产生真实的收益。
